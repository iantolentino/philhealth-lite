
# 🇵🇭 PhilHealth Lite — Simple Health Guide

PhilHealth Lite is a simple, beginner-friendly web guide that explains PhilHealth in the Philippines in an easy-to-understand way.  
It is designed for everyday users who want clear information about membership, contributions, benefits, and how to claim.

> ⚠️ Note: This is NOT an official PhilHealth website. This is an informational guide only.

---

## 🌐 Live Features

This website includes:

- 👥 Who is covered under PhilHealth
- 📝 How to register (Employed, Self-employed, OFW, Voluntary, Senior Citizen)
- 💰 Contribution calculator and rate table
- 💊 Full list of benefits (hospitalization, maternity, dental, dialysis, etc.)
- 🏥 Covered illnesses and conditions
- 📄 Step-by-step guide on how to claim benefits
- 🤝 YAKAP primary care program guide
- 🔗 Official PhilHealth links and hotlines
- ❓ Frequently Asked Questions (FAQ)
